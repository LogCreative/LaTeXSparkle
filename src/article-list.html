第三节 舒适的 \(\rm \LaTeX\) 编辑环境

如何通过 VS Code 以及 Markdown 文档预览 \(\rm \LaTeX\) 公式。并尝试使用这样的流程写一次数学作业，最后将演示如何使用 Overleaf 与本地双向同步。

第四节 使用 TikZEdt 绘制 TikZ 矢量图形
状态：文稿准备中

手动编写 \(\rm \LaTeX\) TikZ 矢量图形代码太费劲，可以使用 TikZEdt 软件即时地看到图形结果。精致的 TikZ 图形使得文档设计语言统一。

第五节 \(\rm \LaTeX\) 中的几何画板 tkz-euclide
状态：文稿准备中

平面几何图形的绘制，在很多情况下都是比较麻烦的，一般需要借助几何画板这类的工具精确绘制。幸运的是，\(\rm \LaTeX\) 提供了 tkz-euclide 宏包用于绘制欧氏几何图形。

第六节 简约统计图
状态：文稿准备中

现在各种各样的数据科学工具可以产生非常精确的统计图表，但是大多数都有一定的个性化局限性。而 \(\rm \LaTeX\) 中的 PGFPlots 宏包可以产生精确又精致的自定义简约统计图，适用于印刷又赏心悦目。

第七节 学术幻灯片
状态：文稿准备中

PowerPoint 仍然需要手动排版。而程式化地生成专业学术 PPT，非 Beamer 宏包莫属。\(\rm \LaTeX\) 专业的排版技术可以原生地生成专业的 PPT，并且可以使用 \(\rm \LaTeX\) 中所有的其他宏包用于生成规整的排版区块。
