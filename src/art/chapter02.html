2.2 各种方式

三屏展示三种方式，抽取式

大约有三种方式在标准视频帧内原生地展示 \(\rm \LaTeX\) 公式，分别为 Beamer, PowerPoint, Manim。

Beamer 语法页面与样式页面

2.2.1 \(\rm \LaTeX\) 所推荐的原生方式是使用 Beamer 宏包所提供的预设模板，以frame环境分割每一个幻灯片，最终通过 PDF 格式保存。但是这种方式所能使用的模板很是有限，并且默认模板样式上有些过时了，这种正式版式更适合于正式场合的讲演，展示方式上可能不是那么的有趣。如果真的只是追求 \(\rm \LaTeX\) 公式插入的话，还是没有 VS Code 插件 Marp 方便。

Manim Kindergarten 相关视频、代码页、转换样例视频

2.2.3 Manim (Mathematical Animation Engine) 是一个用于制作视频的动画引擎，它用 Python 编程的方式创建精确的动画。有兴趣学习的小伙伴可以前往国内 Manim Kindergarten 创作社区查看详情。用 Python 编程确实挺好玩的，就是配置和学习起来不是特别容易，里面确实有许多好玩的动态效果，比如形状转换动画、文字的描边淡出效果、三维数学动画，更加适合视频这种类别。

老片段

2.2.2 IguanaTeX 就是一款能在幻灯片随地插入 \(\rm \LaTeX \)公式的 PowerPoint 插件。配合 Office 365 的平滑切换就能够像模像样。

2.3 配置

2.3.1 前往网站下载 IguanaTeX 加载项，Mac 用户有一个第三方版本。安装Office 365 或 2019 以使用更新的功能。前往 TeX2img 下载页下载 TeX2img 软件，解压即可。

2.3.2 让插件常驻 PowerPoint。打开 PowerPoint，进入文件选单，点选选项，在加载项选项卡中，管理 PowerPoint 加载项并转到，添加 IguanaTeX 加载项。完成后关闭。这样每次打开 PowerPoint 的时候就能看到这个插件了。

2.3.3 在开始之前，还是需要先安装 \(\rm \LaTeX \) 的！通过 Mik\(\rm \TeX\) 官网 下载 Mik\(\rm \TeX\)，按照流程安装即可。

高级弹幕，跳转链接。

2.4 插入公式

2.4.1 New LaTeX display 创建新的\(\rm \LaTeX \) 公式。在中间的空白区域输入一些文字，比如：\LaTeX Sparkle，如果一切顺利的话，单击Generate按钮就可以生成了。

头秃时刻：\LaTeX 可以生成专有符号 \(\rm \LaTeX \) 。

备注：如果不顺利，可以勾选Debug模式，根据 MikTeX 引导安装对应的包即可。

哦，好像少了个空格，\(\rm \LaTeX \) 是吞空格的，选中后单击Edit LaTeX display编辑 \(\rm \LaTeX \) 公式，在中间加入一个波浪线就可以了！

头秃时刻：~ 代表空格。

2.4.2 而我们一般都需要使用公式环境，就在正文创建一个equation环境，在其中加入公式，例如：a = b + c，单击ReGenerate按钮。

头秃时刻：equation 环境专门用于生成公式，使用 \begin 和 \end 定界符包住公式区。 
\begin{equation}
公式区
\end{equation}

2.4.3 然而却多出现了一个公式标号，不太爽，可以通过添加无编号标记去除，切走公式区，我们就可以就现状单击Make Default按钮存储为默认状态了。这样，下次再创建公式时，就可以使用当前的模板了。

头秃时刻：\nonumber 为无编号标记。

2.5 公式矢量化

2.5.1 首先需要定向软件地址。打开Main Settings设置选单，设定 TeX2imgc 的位置，点击右边的...选择下载好的 TeX2imgc.exe（不带图标的可执行程序），点击Ok生效。

2.5.2 点选已经生成的公式，点击Vectorize selection矢量化选择项。完成后，右键整体，取消组合，就可以将单个字符提取出来了。

备注：组合下的元素在后期动画时会有一定的缩放问题。

2.6 公式平滑切换

2.6.1 下面我们通过模仿一个第 5 节的片段来讲述采用 PowerPoint 的平滑切换功能展示公式变化。

2.6.2 新建一个公式，在公式环境键入

PV = \frac{2}{3} N \left \langle \frac{1}{2} mv^2 \right \rangle
\[ PV = \frac{2}{3} N \left \langle \frac{1}{2} mv^2 \right \rangle \]
头秃时刻：\frac{分子}{分母} 用以生成分数。\langle,\rangle用以生成左右尖角括号，用以表示平均数。添加\left,\right定界符是为了限定括号区域，从而拉伸括号。

生成完毕后，点击公式矢量化按钮，并取消组合。

同样的方法，

\left \langle \frac{1}{2} mv^2 \right \rangle = \frac{3}{2}kT
\[ \left \langle \frac{1}{2} mv^2 \right \rangle = \frac{3}{2}kT \]

矢量化、取消组合。

然后右击该幻灯片，复制幻灯片，在新幻灯片中，将对应的元素删除，并将元素放入正确的位置上。

接着，打开切换选项卡，在切换效果中选择平滑。

同理，将最后一张幻灯片完成，我们就可以享受成果了！

是不是在几分钟之内就完成了一个精致公式动画的制作了呢？这种方法可以灵活地调整公式的位置，可以说非常方便了。
